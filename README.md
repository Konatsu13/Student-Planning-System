## 🛠 The Technology We Use
- **Design:** Figma (Prototyping)
- **Styling:** Antigravity
- **Framework:** Next JS (PWA/Mobile Web)
- **Database:** Supabase

<details>
<summary><b>Why Supabase for this School Project?</b></summary>

- **Easy To Use:** Sangat cepat untuk setup awal.
- **Fast Performance:** Cocok untuk aplikasi mobile yang butuh akses cepat.
- **Good Security:** Data tugas siswa tersimpan dengan aman.
- **Free Tier:** Sangat ramah di kantong pelajar!
</details>

## 📊 Comparison: Vanilla vs Framework
| Type | Pros | Cons | Examples |
| :--- | :--- | :--- | :--- |
| **Vanilla** | Fundamental / Already learned | Slower development time | HTML, CSS, JS, PHP |
| **Framework** | Faster development | Requires deep JS understanding | Next JS, Node JS, TypeScript |
